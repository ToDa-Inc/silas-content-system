# Video Analysis Criteria - Detailed Guide

This document explains the 5 outlier criteria used to evaluate why a video went viral.

---

## The 5 Core Criteria

Each video is scored 1-10 on each criterion (total 50 points).

| # | Criteria | Weight | What It Measures |
|---|----------|--------|------------------|
| 1 | Instant Hook | 20% | Attention in 0-2 seconds |
| 2 | High Relatability | 20% | "That happened to me" factor |
| 3 | Cognitive Tension | 20% | Curiosity or disagreement |
| 4 | Clear Value | 20% | Insight, script, or framework |
| 5 | Comment Trigger | 20% | Discussion potential |

---

## Criterion 1: Instant Hook (0-2 seconds)

**Definition:** The video captures attention within the first 0-2 seconds.

### What to Look For

| Element | Description | Examples |
|---------|-------------|----------|
| Time-specific | Exact time mentioned | "Freitag 16:55", "Montag 9:01" |
| Situation-specific | Concrete scenario | "Boss sends urgent email" |
| POV language | "You/your" language | "Your bag is packed" |
| Conflict word | Urgency/problem word | "Dringend", "Warning", "Stop" |
| Visual hook | Eye-catching visual | Bright color, movement |

### Scoring

| Score | Description |
|-------|-------------|
| 8-10 | Multiple hook elements present |
| 5-7 | Some hook elements |
| 1-4 | Weak or missing hook |

### Examples

**High score (10/10):**
> "Es ist Freitag, 16:55 Uhr: Deine Tasche ist gepackt, und du bist bereit, das Büro zu verlassen. Plötzlich kommt eine E-Mail mit dem Betreff 'Dringend' von deinem Chef."

- Time-specific ✓
- Situation-specific ✓
- POV language ✓
- Conflict word ✓

---

## Criterion 2: High Relatability

**Definition:** The viewer immediately thinks "That happened to me" or "I know this feeling."

### What to Look For

| Element | Description |
|---------|-------------|
| Universal workplace situation | Something most people experience |
| Common frustration | Everyday pain points |
| No abstract concepts | Concrete, specific scenarios |
| Emotional trigger | Taps into real feelings |

### Scoring

| Score | Description |
|-------|-------------|
| 8-10 | Highly relatable, universal situation |
| 5-7 | Somewhat relatable |
| 1-4 | Niche or abstract |

### Examples

**High score:**
- "Your boss sends urgent email on Friday at 5pm"
- "Someone talks behind your back in a meeting"
- "You get asked to do more work right before leaving"

**Low score:**
- "Corporate communication theory"
- "The principles of leadership"

---

## Criterion 3: Cognitive Tension

**Definition:** The video creates curiosity or disagreement, making the viewer want to know more.

### What to Look For

| Element | Description |
|---------|-------------|
| Wrong → Right pattern | Shows what NOT to do before solution |
| Question at end | Forces cognitive processing |
| Conflict words | Creates tension |
| Incomplete information | Teases without revealing |

### The Zeigarnik Effect

The brain remembers incomplete tasks better than complete ones. By showing the problem WITHOUT the solution, the viewer is psychologically compelled to continue (read the caption).

### Scoring

| Score | Description |
|-------|-------------|
| 8-10 | Strong tension, multiple elements |
| 5-7 | Some tension |
| 1-4 | No tension |

---

## Criterion 4: Clear Value

**Definition:** The viewer gains something tangible from watching.

### What to Look For

| Element | Description |
|---------|-------------|
| Actionable script | Exact words to use |
| Framework | Structured approach |
| Specific phrases | Copy-paste ready |
| Validation | Confirms their feelings |

### Scoring

| Score | Description |
|-------|-------------|
| 8-10 | Full script or framework provided |
| 5-7 | Some value |
| 1-4 | No clear value |

### Examples

**High value:**
- "4 responses you can use: 1) I can help but not in this timeframe..."
- "Red flags: 1) They only talk to you when they need something"

**Low value:**
- "Communication is important"
- "Be more confident"

---

## Criterion 5: Comment Trigger

**Definition:** The video creates discussion or engagement.

### What to Look For

| Element | Description |
|---------|-------------|
| Direct question | Asks viewer to respond |
| Controversial statement | Might spark debate |
| Validation | "You know this feeling" |
| CTA | Clear call to action |

### Scoring

| Score | Description |
|-------|-------------|
| 8-10 | Multiple engagement triggers |
| 5-7 | Some engagement |
| 1-4 | No engagement trigger |

## Video Type Adaptations

### Text Overlay Videos (Short-Form)

For these videos, criteria 1-3 are primarily assessed from the TEXT OVERLAYS in the frames.

| Criterion | How to Assess |
|-----------|---------------|
| Instant Hook | First frame text |
| High Relatability | Text describes universal situation |
| Cognitive Tension | Question at end, "wrong answers" shown |
| Clear Value | In caption |
| Comment Trigger | Question in caption or on frames |

### Talking Head Videos (Long-Form)

For these videos, criteria are assessed from the TRANSCRIPT + VISUAL.

| Criterion | How to Assess |
|-----------|---------------|
| Instant Hook | First 3 seconds of speech |
| High Relatability | Topic and scenario in transcript |
| Cognitive Tension | Questions, conflict words in speech |
| Clear Value | Full script or framework in transcript |
| Comment Trigger | CTA in speech |

## Total Score Interpretation

| Score | Rating | Action |
|-------|--------|--------|
| 40-50 | ✅ Highly Replicable | Blueprint found |
| 30-39 | ✅ Strong Pattern | Adapt for niche |
| 20-29 | ⚠️ Moderate | Analyze further |
| <20 | ❌ Weak | Not a strong outlier |

## Common Patterns Found

### Pattern 1: Time + Situation + Conflict

```
[TIME] + [SITUATION] + [CONFLICT]
"Freitag 16:55" + "Dein Chef schickt E-Mail" + "Dringend"
```

### Pattern 2: Wrong → Right

```
[SHOW 3 WRONG ANSWERS] → [IMPLY RIGHT]
❌ "Sorry, ich muss gehen"
❌ "Nein, geht nicht"
❌ (Solution in caption)
```

### Pattern 3: Question at End

```
[CONTENT] + [QUESTION]
"Das ist die smart Antwort..." + "Oder?"
```
