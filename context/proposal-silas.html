What We're Building

A content intelligence and production system for Instagram Reels — built for info product creators managing multiple clients and niches.

The system does the 80%: research, pattern analysis, content strategy, hook and script generation, video processing (including multiple layout formats), and thumbnail creation. You do the 20%: review and approve content in the dashboard, then upload to Instagram and add trending audio.

The existing B-roll workflow is preserved. The system takes your B-roll, generates the text overlays, burns them into the video programmatically, and outputs a ready-to-upload file. From any viral outlier (scraped or pasted URL), the AI extracts structure and intent — then you choose how to execute it using up to five preset video formats (layout pipelines, not five separate AI products).

Built for one client at launch. Multi-client from day one.

Deliverables by Week

Week 1 — Intelligence Layer + Dashboard

What gets delivered:

  • Web dashboard live and accessible from any browser — no installation required
  • Competitor profile scraper: pulls top-performing Reels from up to 10 accounts per client on demand, ranked by engagement score
  • Outlier detection: flags posts performing 5–10x above an account's average, with exact outlier ratio shown per post
  • Manual viral reference input: paste any public Reel URL and the system scrapes, analyzes, and adds it to the intelligence layer — instant research on any post you find interesting
  • AI analysis per outlier: hook type, emotional trigger used, caption structure, why it likely performed — specific to each post, not generic observations
  • Niche patterns document per client: aggregated across all analyzed posts, updated automatically on every new scrape or manual input
  • First client fully configured: niche keywords, competitor accounts, ICP, products, tone of voice, pain points, personal stories
Verifiable at end of Week 1: Dashboard is live. Run a scrape or paste a Reel URL. Get back a ranked feed of viral posts with AI analysis per post and an updated patterns document showing what's working in the niche right now.

Week 2 — Content Generation

What gets delivered:

  • Outlier-to-content flow: pick a viral reference (from scrape or pasted URL). AI summarizes what made it work; you then generate hooks, captions, and scripts informed by that specific outlier — not only generic niche patterns
  • Hook generator: 10–20 hook options per run, derived from niche patterns + client context + (optional) selected outlier analysis
  • Caption generator: full Instagram caption per selected hook — mini-story structure, relatable opener, insight, keyword CTA, max 5 hashtags — written in the client's voice
  • Talking head script generator: complete 60-second script structured as hook → situation → 3 insights → conclusion → CTA, adapted to the creator's tone of voice as configured in their profile
  • IG Story text generator: short punchy story text variations based on the same content angle
  • All outputs reviewable and editable in the dashboard before any production step
Verifiable at end of Week 2: Generate a batch of hooks from real niche data. Select one. Get back a complete caption, a talking head script, and story text — all usable without editing, all in the client's voice.

Week 3 — Video Processing + Format Presets + Thumbnail + Post Queue

What gets delivered:

  • Five output format presets — predefined layout pipelines (same B-roll + captions, different frame treatment). Examples covered in this build:
    • Full-frame B-roll (9:16) — standard Reel: video fills the frame, text overlay on top
    • Static / still-first — optimized for still or minimal-motion source (single strong frame + text; can pair with carousel flow elsewhere)
    • Letterbox / cinematic — B-roll scaled with black bars top and bottom; hook text can sit in bars or on picture per preset rules
    • 4:5 center in 9:16 — pillarboxed look: content centered with side bars (common feed aesthetic)
    • Hook-first / minimal on-video text — lighter on-screen copy; fuller story lives in caption (reduces clutter on busy B-roll)
    After analyzing an outlier, the system suggests a recommended format based on structure (you can always override). Switching format re-runs the same assets through a different template — not a full re-shoot.
  • B-roll video processing: upload your existing B-roll (pre-cut). The system burns the generated text overlay onto the video and exports a ready-to-upload .mp4 per selected format
  • Text overlay parameters configurable per client: default position, font size, line breaks, style — applied consistently; per-post overrides where needed
  • Revision prompt box: type natural-language instructions (e.g. "shorter hook", "move text higher", "bolder outline", "two lines only") — system updates overlay copy and/or styling and re-exports the video. This covers the same class of edits as tweaking text in Instagram, but with consistent brand rules, version history, and batch-friendly workflow across many clients. It does not replace Instagram for a one-word tweak on a single post — it wins on volume, consistency, and repeatability.
  • Thumbnail / cover image generator: AI-generated cover images per Reel (gpt-image-1.5, photorealistic, portrait format) — for the Reel cover on the profile grid
  • Post queue: video preview + caption + hashtags + thumbnail. Approve, reject, revision prompt, or regenerate thumbnail
  • Download package per post: video file + caption + hashtags — post manually in Instagram (add trending audio), typically under ~2 minutes
Verifiable at end of Week 3: From a chosen outlier, generate hooks and caption, pick a format preset (or accept the recommendation), upload B-roll, export video with burned-in text, optionally use the revision prompt to iterate, generate thumbnail, download package. Ready to post.

Week 4 — Auto-Learning Loop + Multi-Client + Handover

What gets delivered:

  • Performance tracking: enter or connect published post metrics (views, likes, comments, saves) — tracked per post per client in the dashboard
  • Auto-learning loop: posts that hit above-threshold performance are added to the winning examples pool — future generation runs are weighted toward what actually worked for that specific client's audience
  • Approval signal learning: approve/reject decisions on hooks and content train the system's understanding of preferences over time
  • Diagnostic framework: posts automatically classified (scale it / adjust hook / adjust CTA / retire) based on view and engagement thresholds
  • Second client live: configured in under 10 minutes — own niche, competitors, ICP, tone, products
  • Handover: complete walkthrough of every part of the system, documented — fully independently operable from day one
Verifiable at end of Week 4: Second client added from scratch live during handover. Full system running for both. Auto-learning loop active — generation quality improves measurably after 3–4 weeks of performance data.

What the System Does NOT Do

  • Does not edit B-roll — video cutting, scene editing, and transitions are out of scope; B-roll must be ready before uploading
  • Does not post to Instagram automatically — Instagram's API does not support Reel scheduling with audio selection; all posting is manual (download package, upload to Instagram, add trending audio, publish)
  • Does not record video — talking head scripts are generated; the creator films themselves
  • Does not manage DMs or comments — content creation and preparation only
  • Does not cover TikTok or YouTube as output platforms — Instagram only in this version
  • Does not make content decisions autonomously — all posts require approval before anything is downloaded or published
